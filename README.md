# MovieList - React приложение для управления списком фильмов

![React](https://img.shields.io/badge/React-18.2.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)

Приложение для просмотра и управления списком фильмов с возможностью добавления в избранное.

## 🚀 Функциональность

- 📽️ Просмотр списка фильмов с детальной информацией
- ❤️ Добавление/удаление фильмов в избранное
- 🔍 Поиск фильмов по названию
- 💾 Автосохранение в localStorage
- 📱 Адаптивный дизайн

# Установка зависимостей
npm install

# Запуск приложения
npm start
\`\`\`

Приложение будет доступно по адресу: \`http://localhost:3000\`

## 🎯 Использование

1. **Просмотр фильмов**: Все фильмы отображаются в виде карточек
2. **Добавление в избранное**: Нажмите на сердечко 🤍 чтобы добавить фильм в избранное (станет ❤️)
3. **Поиск**: Используйте поисковую строку для фильтрации фильмов по названию
4. **Сохранение**: Избранные фильмы автоматически сохраняются в браузере
